# -Linear-regression-for-house-price-prediction

一个基于房屋销售数据的机器学习入门项目。  
本项目使用 **Jupyter Notebook** 完成数据清洗、探索性分析、特征工程与 **多元线性回归建模**，用于研究房屋特征与销售价格之间的关系，并对未知房价进行预测。

---

## 项目简介

房价预测是回归分析中的经典问题之一。  
本项目基于已有房屋销售价格及房屋属性数据，分析影响房价的关键因素，例如：

- 房屋面积
- 房间数量
- 建造年份
- 地理位置或区域特征
- 其他房屋属性变量

通过建立线性回归模型，探索各特征与房价之间的线性关系，并将训练得到的模型应用到未知样本上，完成房价预测。

---

## 项目目标

本项目主要完成以下内容：

1. 读取并理解房价数据集
2. 对数据进行清洗与预处理
3. 完成探索性数据分析（EDA）
4. 分析特征变量与房价之间的关系
5. 构建多元线性回归模型
6. 对未知样本进行房价预测
7. 评估模型拟合效果并解释结果

---

## 数据文件说明

项目中包含以下数据文件：

- `房价.csv`：原始房价数据
- `house_price_simple.csv`：用于建模分析的简化数据
- `house_price_predict_simple.csv`：用于预测的未知样本数据
- `使用线性回归房价预测Data.ipynb`：项目主分析 notebook

---

## 项目流程

### 1. 数据读取
使用 Pandas 读取房价数据集，查看字段名称、数据类型、样本数量和整体结构。

### 2. 数据预处理
对原始数据进行清洗，包括：

- 检查缺失值
- 处理异常值或不合理值
- 筛选用于建模的关键字段
- 转换数据类型
- 构建适合线性回归分析的数据表

### 3. 探索性数据分析
通过可视化和描述统计分析数据特征，包括：

- 数值变量分布情况
- 房价分布情况
- 房屋面积、房间数等变量的分布
- 不同类别变量对房价的影响
- 特征变量与售价之间的散点关系

### 4. 建立线性回归模型
基于多个房屋属性变量建立多元线性回归模型，分析不同特征对房价的影响方向和影响程度。

### 5. 模型结果分析
对模型输出结果进行解释，包括：

- 回归系数
- 各变量显著性
- 模型整体拟合效果
- 特征与房价之间的线性关系是否明显

### 6. 未知样本预测
对 `house_price_predict_simple.csv` 中的房屋数据进行预测，得到对应房价结果，用于模拟实际业务中的价格估计任务。
